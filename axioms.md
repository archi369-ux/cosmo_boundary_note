# Axioms & Semantic Rules (v0.3)

## A0. Language/domain axiom
Some questions are **undefined** if answering them requires extending primitives beyond their intended meaning.

## A1. Record-preserving ontology (RP)
**T exists** as a total record structure and is **immutable** unless explicitly modified by a corruption/erasure rule.

## A2. Traversal index
t is a directed traversal index through records in T.

## A3. Rewind
Reversing t is a read operation over T that can recover earlier records.

## A4. Loss
Information is ‘lost’ only if T is corrupted/erased (not merely due to expansion or lack of access).

## A5. Space model S
Space is represented as a 3D coordinate chart (x,y,z) with an origin O used as a reference for pattern description.
